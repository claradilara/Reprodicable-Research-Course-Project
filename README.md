# Reprodicable-Research-Course-Project
Data Science Salaries 2023/2024

This project analyzes the Data Science Salaries 2023/2024 dataset to understand the key factors that influence salary levels in data-related roles. The main objective is to investigate how variables such as experience level, remote work status, job title, and geographic location impact salaries. The dataset is first cleaned and preprocessed to ensure consistency and accuracy, focusing on standardizing salary values, categorizing experience levels, and refining job titles. A regression analysis is then conducted using log-transformed salary as the dependent variable, with relevant dummy variables included to capture categorical effects. Finally, the results are visualized through an interactive dashboard and plots, providing clear insights and interpretations of the findings. The project is a collaborative effort, with team members contributing across data cleaning, statistical modeling, and visualization to deliver a comprehensive analysis.
